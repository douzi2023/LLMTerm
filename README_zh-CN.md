## LLMTerm
LLMTerm 是一个可通过 SSH 访问的虚拟终端，由大型语言模型 (LLM) 驱动。

## 🌐 概述
LLMTerm 允许用户通过 SSH 连接，并与由 AI 模型 (LLM) 驱动的模拟终端环境进行交互。
它的行为类似于普通的 shell——您输入命令，它会做出响应——但您看到的所有内容都是由一个能够理解上下文、命令和用户意图的智能模型生成的。
## ✨ 功能
- 🧠 **基于 LLM 的 shell 模拟** — 响应诸如 `ls`、`cat` 之类的命令或自定义任务。
- 🔐 **SSH 兼容访问** — 使用任何 SSH 客户端连接。
- ⚙️ **可配置的后端** — 可切换模型、调整温度并控制行为。
- 💬 **流式输出** — 体验逼真的命令行反馈。
- 🧩 **可扩展设计** — 轻松集成自定义命令或插件。
## 🧰 使用方法
1. 生成 SSH 主机密钥（如果您还没有）：
```
python ssh-keygen.py
```
2. 启动服务器：
```
python3 main.py
```
3. 从另一个终端连接：
```
ssh -p 2222 root@localhost
```
您将进入一个 AI 模拟的 shell 环境。
尝试以下命令：
```
ls
cat /etc/motd
echo "Hello from the matrix"
```
## 🧩 架构
```
┌─────────────┐

│ SSH 客户端   │ ←→ AsyncSSH 服务器 ←→ LLM 引擎 ←→ 响应生成器

└─────────────┘
```
SSH 层处理安全连接，而 LLM 引擎处理用户输入并生成智能响应。
